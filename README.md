# Spotify Clone - Ứng dụng Nghe nhạc Trực tuyến

Chào mừng bạn đến với dự án **Spotify Clone**! Đây là một ứng dụng nghe nhạc web đơn giản được xây dựng hoàn toàn bằng **HTML, CSS và JavaScript thuần (Vanilla JS)** mang phong cách thiết kế hiện đại của Spotify.

![Spotify Clone Demo](demo.png)

## Tính năng nổi bật

- **Trình phát nhạc đầy đủ**: Phát, tạm dừng, chuyển bài tiếp theo, quay lại bài trước.
- **Chế độ phát**: Hỗ trợ phát ngẫu nhiên (Shuffle) và lặp lại (Repeat).
- **Giao diện Responsive**: Tối ưu hóa trải nghiệm trên cả máy tính (Desktop) và điện thoại (Mobile).
- **Tìm kiếm thông minh**: Tìm kiếm bài hát theo tên hoặc nghệ sĩ trong thời gian thực.
- **Lời chào theo thời gian**: Hiển thị lời chào "Chào buổi sáng", "Chào buổi chiều" hoặc "Chào buổi tối" dựa trên giờ thực tế.
- **Đĩa nhạc xoay (Disc Rotation)**: Hiệu ứng đĩa nhạc xoay khi đang phát bài hát trong chế độ xem chi tiết.
- **Danh sách hàng chờ (Queue)**: Xem nhanh danh sách các bài hát sẽ phát tiếp theo.
- **Điều khiển âm lượng & Tiến độ**: Thanh trượt điều chỉnh âm lượng và thời gian bài hát mượt mà.

## Công nghệ sử dụng

- **HTML5**: Cấu trúc ứng dụng và các phần tử ngữ nghĩa.
- **CSS3**: Thiết kế giao diện, hiệu ứng Glassmorphism, animations và Flexbox/Grid Layout.
- **Vanilla JavaScript**: Xử lý logic phát nhạc, quản lý trạng thái và tương tác DOM.
- **FontAwesome**: Hệ thống icon phong phú.
- **Google Fonts**: Sử dụng font *Plus Jakarta Sans* và *Inter* mang lại cảm giác cao cấp.

## Cấu trúc thư mục

```text
Gift_SpotifyClone-main/
├── index.html          # File giao diện chính
├── demo.png            # Hình ảnh demo dự án
├── style/
│   ├── style.css       # Style chính cho ứng dụng
│   ├── modal.css       # Style cho giao diện phát nhạc chi tiết
│   ├── script.js       # Toàn bộ logic xử lý nhạc
│   ├── img/            # Chứa các ảnh bìa album
│   └── sound/          # Chứa các file nhạc (.mp3)
└── README.md           # Hướng dẫn dự án
```

## 🛠️ Cách cài đặt và chạy

1. **Tải dự án về máy**:
   ```bash
   git clone https://github.com/DrGifter/Gift_SpotifyClone.git
   ```
2. **Mở dự án**:
   Mở file `index.html` trực tiếp trên trình duyệt của bạn (Chrome, Edge, Firefox, etc.).
3. **Thưởng thức**:
   Trải nghiệm các tính năng nghe nhạc và tìm kiếm các bài hát yêu thích.
